# 범용 블로그 원고 자동 생성 프롬프트 (개선 버전)

## 기본 구조
- **메인 키워드**: 제공된 블로그 가이드에서 메인 키워드 확인 및 사용
- **서브 키워드**: 제공된 세부 키워드 목록에서 선택
- **업체명**: 제공된 업체명 사용
- **글자 수**: 제공된 글자 수 기준 (예: 1800자 이상)
- **섹션 수**: 이미지 수에 맞춰 조정 (번호는 0부터 시작)
- **글 유형**: 제공된 유형 (예: 정보성 글)

## 원고별 차별화 전략 (핵심)

### 1. 스토리텔링 접근법 다양화 (20가지)
- **개인 경험형**: 개인 실패 경험 → 재도전 과정 → 성공 사례 발견
- **추천 기반형**: 지인 추천 → 설명회 참석 → 결정 과정
- **문제 해결형**: 기존 사업 한계 → 대안 탐색 → 해결책 발견
- **시장 분석형**: 업계 동향 관찰 → 브랜드 비교 → 선택 이유
- **우연 발견형**: 우연한 발견 → 심층 조사 → 확신 과정
- **전문가 시각형**: 업계 전문성 → 객관적 분석 → 추천 결론
- **투자자 관점형**: 투자 검토 → 수익성 분석 → 투자 결정
- **변화 추구형**: 현재 상황 불만 → 변화 의지 → 새로운 시도
- **성장 지향형**: 더 큰 꿈 → 기회 탐색 → 성장 동력 발견
- **안정성 중심형**: 안정적 사업 필요 → 검증된 모델 탐색 → 확신
- **혁신 추구형**: 차별화 필요성 → 혁신적 모델 발견 → 도전 결심
- **효율성 중심형**: 효율적 운영 추구 → 시스템 분석 → 최적 선택
- **미래 지향형**: 미래 트렌드 예측 → 선제적 준비 → 앞서가는 선택
- **가족 중심형**: 가족을 위한 선택 → 안정성 검토 → 확신 과정
- **커뮤니티형**: 온라인 정보 수집 → 후기 분석 → 결정 과정
- **데이터 기반형**: 객관적 데이터 수집 → 분석 결과 → 논리적 결정
- **직감 기반형**: 첫인상과 직감 → 확인 과정 → 직감 검증
- **비교 분석형**: 여러 옵션 비교 → 장단점 분석 → 최종 선택
- **단계적 검증형**: 1차 관심 → 2차 검증 → 3차 확신 → 최종 결정
- **목표 달성형**: 구체적 목표 설정 → 수단 탐색 → 최적 방법 발견

### 2. 정보 제시 순서 패턴화 (20가지)
- **패턴 1**: 개인적 배경 → 브랜드 소개 → 시스템 장점 → 수익성 → 성공사례 → 결론
- **패턴 2**: 브랜드 특징 → 개인적 관심 → 운영 방식 → 지원 시스템 → 실제 사례 → 마무리
- **패턴 3**: 문제 인식 → 해결책 탐색 → 브랜드 발견 → 차별점 → 투자 가치 → 결정
- **패턴 4**: 업계 현황 → 브랜드 차별성 → 개인적 선택 → 시스템 분석 → 미래 전망 → 정리
- **패턴 5**: 경험담 → 비교 분석 → 핵심 장점 → 성공 요인 → 실증 사례 → 추천
- **패턴 6**: 시장 트렌드 → 기회 포착 → 브랜드 만남 → 가능성 확인 → 투자 결정 → 기대
- **패턴 7**: 고민 상황 → 정보 수집 → 브랜드 발견 → 심층 분석 → 확신 과정 → 결론
- **패턴 8**: 브랜드 첫인상 → 호기심 증가 → 상세 조사 → 장점 확인 → 사례 검증 → 추천
- **패턴 9**: 현재 상황 → 변화 필요성 → 대안 모색 → 브랜드 선택 → 기대 효과 → 다짐
- **패턴 10**: 목표 설정 → 방법 탐색 → 브랜드 분석 → 적합성 검토 → 성공 가능성 → 결정
- **패턴 11**: 업계 분석 → 성공 요인 분석 → 브랜드 발견 → 경쟁력 확인 → 미래성 → 선택
- **패턴 12**: 개인 니즈 → 해결책 탐색 → 브랜드 특장점 → 시스템 분석 → 확신 → 추천
- **패턴 13**: 시장 기회 → 성공 모델 탐색 → 브랜드 만남 → 차별화 요소 → 성장성 → 투자
- **패턴 14**: 변화 동기 → 옵션 비교 → 브랜드 우위 → 지원 체계 → 성공 사례 → 확신
- **패턴 15**: 문제 의식 → 해결 방향 → 브랜드 솔루션 → 실행 방법 → 예상 결과 → 결심
- **패턴 16**: 업계 경험 → 전문적 분석 → 브랜드 평가 → 투자 가치 → 성공 전망 → 추천
- **패턴 17**: 현실 인식 → 기회 탐색 → 브랜드 검토 → 가능성 평가 → 리스크 분석 → 도전
- **패턴 18**: 성공 열망 → 모델 연구 → 브랜드 발견 → 시스템 검증 → 성과 예측 → 시작
- **패턴 19**: 안정 추구 → 검증된 모델 탐색 → 브랜드 신뢰성 → 지원 시스템 → 안정성 → 선택
- **패턴 20**: 혁신 의지 → 차별화 모델 탐색 → 브랜드 혁신성 → 독창성 분석 → 경쟁우위 → 결정

### 3. 핵심 내용 분배 전략
**블로그 가이드의 "반드시 들어가야 할 내용"을 그룹별로 분배**:

**그룹 A (수익성/경제성 중심)**:
- 수익 관련 수치와 성공 사례
- 비용 절감 요소 (인건비, 운영비 등)
- 가격 정책 및 마진 구조

**그룹 B (운영 편의성 중심)**:
- 운영 시스템과 프로세스
- 인력 관리 및 교육
- 공급망 및 지원 체계

**그룹 C (차별성/기술력 중심)**:
- 독특한 기술이나 노하우
- 경쟁 우위 요소
- 브랜드 고유 특성

**그룹 D (성장성/안정성 중심)**:
- 확장 사례 및 성장 지표
- 본사 지원 및 시스템
- 시장에서의 입지

**그룹 E (고객 만족/서비스 중심)**:
- 고객 대상 혜택 및 서비스
- 만족도 관련 요소
- 재방문 및 추천 요소

**그룹 F (신뢰성/전문성 중심)**:
- 대표자/창립자 배경 및 경력
- 회사 역사 및 실적
- 업계 평판 및 인지도

**그룹 G (접근성/편의성 중심)**:
- 창업 조건 및 요구사항
- 교육 및 지원 프로그램
- 금융 지원 및 혜택

*각 원고마다 1-2개 그룹을 선택하여 해당 내용을 중심으로 구성*

### 5. 연결어 다양화 (40가지)
**인과관계**: 덕분에, 때문에, 그래서, 따라서, 그 결과, 이로 인해, 이것이 바로, 그러므로, 이에 따라, 결과적으로
**추가설명**: 게다가, 또한, 더불어, 뿐만 아니라, 무엇보다, 특히, 그중에서도, 더 나아가, 이와 함께, 동시에
**대조**: 하지만, 그러나, 반면에, 대신에, 그런데, 다만, 오히려, 역으로, 반대로, 그와 달리
**강조**: 정말, 무척, 굉장히, 상당히, 매우, 특별히, 확실히, 분명히, 실제로, 사실상

## 성공사례 활용 다양화 (범용화)

### 1. 사례 제시 관점 (15가지)
1. **빠른 성과 중심**: "단기간 내 놀라운 결과"
2. **업종 전환 중심**: "기존 사업에서 성공적 전환"
3. **확장 성공 중심**: "1호점 성공 후 연속 확장"
4. **초보자 성공 중심**: "경험 없어도 가능한 성공"
5. **전문가 선택 중심**: "업계 전문가도 인정한 선택"
6. **수익성 중심**: "예상을 뛰어넘는 수익 창출"
7. **만족도 중심**: "기대 이상의 만족스러운 결과"
8. **안정성 중심**: "꾸준하고 안정적인 성과"
