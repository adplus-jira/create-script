# 블로그 원고 자동 생성 프롬프트

## 기본 구조
- **메인 키워드**: 제공된 블로그 가이드에서 메인 키워드 확인 및 사용
- **서브 키워드**: 제공된 세부 키워드 목록에서 선택
- **업체명**: 제공된 업체명 사용
- **글자 수**: 제공된 글자 수 기준 (예: 1800자 이상)
- **섹션 수**: 이미지 수에 맞춰 조정 (번호는 0부터 시작)
- **글 유형**: 제공된 유형 (예: 정보성 글)

## 형식 규칙
- **제목**: 메인 키워드 + 서브 키워드 + 업체명 조합으로 자연스러운 제목
  - 예: "[메인 키워드] [서브 키워드] [업체명], [특징/혜택] 있어요"
- **문장 구조**: 
  - 각 문장이 끝나면 반드시 개행
  - 한 줄에 최대 18자 내외로 제한하여 가독성 향상
  - 예시:
    ```
    [메인 키워드]를 고민하면서
    가장 걱정됐던 건 본사의 지원
    시스템이었어요.
    ```
- **메인 키워드**: 본문에 **반드시 7-8회** 자연스럽게 반복
- **각 섹션**: 숫자(0, 1, 2...)로 시작하고 짧은 단락으로 구성

## 말투 및 어조
- **친근하고 자연스러운 블로그 말투 사용**:
  - "~해요", "~이에요", "~것 같아요" 등의 친근한 종결어 사용
  - "~네요", "~랍니다", "~답니다"와 같은 어색한 종결어는 지양
  - "도움이 되죠", "효과가 있죠"처럼 가르치는 듯한 말투 사용 금지
- **적절한 구어체**: "~하더라고요", "~라고 해요"와 같은 자연스러운 구어체 사용
- **감정 표현**: 개인적인 생각이나 감정을 자연스럽게 표현 (예: "정말 놀라웠어요", "이런 시스템이 있으면 얼마나 편할까요")
- **물음표/느낌표**: 자연스러운 위치에 필요할 경우에만 최대 3개까지만 사용

## 내용 구성
- **도입부**: 개인적인 경험이나 관심을 바탕으로 자연스럽게 주제 소개
- **본문 구성**: 
  - 내용을 먼저 자연스럽게 서술하고, 적절한 길이마다 이미지 번호를 배치하여 글을 분리
  - 핵심 장점/정보를 다양한 순서로 배치하여 원고마다 다른 흐름 형성
  - 제공된 필수 내용을 모두 한 원고에 넣지 말고 선택적으로 포함
- **마무리**: 간결하게 정리하고 링크만 제공 (설명 문구 없이)

## 내용 요소 및 성공 사례
- 블로그 가이드에서 제공하는 **"반드시 들어가야 할 내용"**과 **"성공사례"**를 참고
- 모든 내용을 하나의 원고에 넣지 말고, 원고마다 다른 내용 요소와 사례를 선택적으로 포함
- 내용 요소의 핵심을 이해하고 자연스러운 말투로 표현
- 성공 사례를 활용할 때 제공된 정보만 사용하고 임의로 내용 추가하지 않기

## 원고 작성 지침
1. 제목은 메인 키워드 + 서브 키워드 + 업체명 조합으로 자연스럽게 구성
2. 제목 바로 다음에 0번부터 시작하여 개인적 경험이나 관심으로 도입
3. 내용을 자연스럽게 서술하고 적절한 길이마다 이미지 번호(0, 1, 2...)를 배치
4. 친근하고 자연스러운 블로그 말투 사용 (어색한 종결어 지양)
5. 메인 키워드를 본문에 7-8회 자연스럽게 포함
6. 블로그 가이드에서 제공한 내용 요소와 성공 사례를 선택적으로 활용하여 원고마다 다른 내용 구성
7. 물음표나 느낌표는 자연스러운 위치에 최대 3개까지만 사용
8. 마지막에는 제공된 링크만 간략히 제공

## 중요 주의사항
- 블로그 가이드에서 제공하는 정보만 사용하고 임의로 내용 추가하지 않기
- 메인 키워드 개수(7-8회)를 정확히 지키기
- 항상 제목 바로 다음에 0번 섹션부터 시작하기
- 제공된 링크 정보 정확히 포함하기
- 매 원고마다 다른 내용 구성으로 중복 피하기
